# Portfolio Website

My personal developer portfolio — built with pure HTML, CSS, and JavaScript.

## Live Demo

🌐 **[spashuev.github.io/portfolio-website](https://spashuev.github.io/portfolio-website)**

## Features

- Clean, responsive design that works on all devices
- Typing animation in the hero section
- Scroll-triggered card animations
- Mobile-friendly navigation
- No frameworks — just vanilla HTML/CSS/JS

## How to Run Locally

```bash
git clone https://github.com/spashuev/portfolio-website.git
cd portfolio-website
# Open index.html in your browser, or use Live Server in VS Code
```

## How to Deploy (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main** branch
4. Your site goes live at `https://spashuev.github.io/portfolio-website`
